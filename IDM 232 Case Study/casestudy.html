Overview

This project was designed for my IDM 241 class at Drexel University. In the project I was assigned the task to create a dynamic recipe website that can perform CRUD operations and can also be viewed by the users. We did research on some PHP recipe websites we liked and deconstructed them to understand how does a databse function in a dynamic website. After the research I worked with PHP and MySQL to create my own recipe website that can show various recipes

Context and Challenge

As mentioned before, this project was done for my IDM 241 class and was supposed to be completed in a 10 week period of time. The languages and tools we used for the project were PHP,HTML, CSS, and MySQL. Ultimately, I wanted to use this class to design something that I could use in my portfolio as well. I had a lot of projects in my portfolio and wanted to do something that lets a user take a quick overview of my skills in backend management and proficiency with PHP and MySQL.

The Problem

The problem was the new language of PHP and MySQL. I had never worked on a database management software, I also did not have any experience with PHP. Other than having no experience in backend programming, debugging was the biggest and most time consuming problem as the errors often did not tell me the root cause and I had to manually get and scrutinize each line of code to find errors

DESIGN GOALS

My Goal was to create a dynamic PHP website that lets a User/Admin perform CRUD operations on any recipe on the website.

Tangible Goals & Metrics Used

The tangible goals were just to create a CRUD application and connecting it to a database.

RESULTS

The results were not perfect but satisfying. I think the project was a success as I was able to create a PHP based Recipe website that allows an Admin to perform CRUD operations and look at the recipes which were my direct goals of this project.

Through my surveys I found 75% of the people liked Website and liked how it offered a short recipes with options to Create, Read, Update and Delete recipes.

Even Though I could not work on uploading a new image I still learned a lot in this class through this project. I realized the importance of setting a realistic goal in the beginning and working on small elements of the a CRUD one by one. I also learned a lot about reusing code from already made examples and other websites and how to tailor it so it fits in my webpage.

In the future, I would love to add an option to edit image and nice animations to every small element of the website I created.
